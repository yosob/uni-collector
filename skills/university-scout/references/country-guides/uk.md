# 英国（UK）搜索策略

## 程序关键词

### 英语（搜索首选语言）
- **工业/产品设计**: Industrial Design, Product Design, Design Engineering
- **交互设计**: Interaction Design, UX Design, User Experience Design, Service Design
- **视觉传达**: Graphic Design, Visual Communication, Illustration, Typography
- **纯艺术**: Fine Art, Contemporary Art, Painting, Sculpture, Printmaking
- **数字媒体**: Digital Media, Media Arts, Animation, Moving Image
- **摄影**: Photography, Photojournalism, Documentary Photography
- **时尚设计**: Fashion Design, Textile Design, Fashion Communication
- **建筑相关**: Architecture, Interior Design, Spatial Design

### 中文（辅助搜索）
- 工业设计, 产品设计, 交互设计, 视觉传达, 纯艺术
- 英国设计大学, 英国艺术院校, UK design university

## 院校类型关键词

- `University` — 综合大学
- `Art School` / `College of Art` — 艺术学院
- `School of Art and Design` — 设计学院
- `Institute` — 研究所/专科学院
- `Conservatoire` — 音乐/戏剧学院（通常排除，不在 schema type enum 中）

## 搜索查询模板

### 按专业搜索
```
"{keyword} course UK university"
"{keyword} masters UK"
"best {keyword} programs UK"
"BA {keyword} university England"
"{keyword} postgraduate UK"
```

### 按城市搜索
```
"design course {city} university"
"art school {city}"
"{keyword} programme {city}"
```

### 按类型搜索
```
"art school design course UK"
"university design programme undergraduate UK"
"best UK art schools ranking"
```

### 组合搜索
```
"UCAS {keyword} course"
"{institution_type} {keyword} UK"
"英国设计大学 {program_cn}"
```

## 英国主要城市（按设计教育密度排序）

**Tier 1**（设计院校密集）: London

**Tier 2**（知名设计院校）: Edinburgh, Glasgow, Manchester, Birmingham, Leeds, Bristol

**Tier 3**（有设计相关专业）: Newcastle, Sheffield, Nottingham, Brighton, Falmouth, Dundee, Cardiff, Oxford, Cambridge, Kingston, Loughborough
