# 美国（US）搜索策略

## 程序关键词

### 英语（搜索首选语言）
- **工业/产品设计**: Industrial Design, Product Design, Design Engineering
- **交互设计**: Interaction Design, UX Design, User Experience Design, HCI, Human-Computer Interaction
- **视觉传达**: Graphic Design, Visual Communication, Communication Design, Typography
- **纯艺术**: Fine Arts, Studio Art, Contemporary Art, Painting, Sculpture
- **数字媒体**: Digital Media, New Media, Media Arts, Animation, Game Design
- **摄影**: Photography, Photojournalism, Fine Art Photography
- **时尚设计**: Fashion Design, Textile Design, Fiber Arts
- **建筑相关**: Architecture, Interior Architecture, Environmental Design, Urban Design

### 中文（辅助搜索）
- 工业设计, 产品设计, 交互设计, 视觉传达, 纯艺术
- 美国设计大学, 美国艺术院校, US design school

## 院校类型关键词

- `University` — 综合大学（含公立/私立）
- `Art School` / `College of Art` — 独立艺术学院
- `School of Design` — 设计学院
- `Institute` — 专业学院（如 Pratt Institute, RISD）
- `College` — 文理学院

## 搜索查询模板

### 按专业搜索
```
"{keyword} program university USA"
"{keyword} MFA program"
"best {keyword} schools US"
"BFA {keyword} program"
"{keyword} graduate program ranking"
```

### 按城市搜索
```
"design school {city}"
"{keyword} program {city}"
"art school {city}"
```

### 按类型搜索
```
"art school design program US"
"best design schools US ranking"
"industrial design program accredited"
```

### 组合搜索
```
"{institution_type} {keyword} program"
"NASAD accredited {keyword} program"
"美国设计大学 {program_cn}"
"top {keyword} schools US"
```

## 美国主要城市（按设计教育密度排序）

**Tier 1**（设计院校密集）: New York, Los Angeles, San Francisco/Bay Area, Chicago

**Tier 2**（知名设计院校）: Providence (RISD), Boston, Philadelphia, Detroit, Seattle, Portland

**Tier 3**（有设计相关专业）: Pasadena, Minneapolis, Columbus, Cincinnati, Atlanta, Austin, Denver, Baltimore, Washington D.C., San Diego, Pittsburgh, St. Louis, Milwaukee

## 认证体系

- **NASAD** (National Association of Schools of Art and Design): 美国艺术与设计院校认证机构
- 搜索 `NASAD accredited` 可以找到经过认证的设计项目
