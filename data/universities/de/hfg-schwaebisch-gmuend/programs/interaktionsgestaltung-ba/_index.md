# Interaction Design (Interaktionsgestaltung)

Interaktionsgestalter*innen entwickeln digitale Produkte und Services. Der Studiengang an der HfG Schwäbisch Gmünd bereitet auf die Gestaltung von Apps, Webseiten, Software und interaktiven Produkten vor.

**Schwerpunkte (ab 4. Semester):**
1. Interface Design - Schnittstellen zwischen Mensch und Computer, Software-Produkte
2. Physical Computing - Tangible Interfaces, Embedded Systems, IoT

**Arbeitsfelder:**
- Application Design - Konzeption, Aussehen und Verhalten von Software-Produkten
- Interface Design - Schnittstellen zwischen Mensch und Computer
- Interaktive Kommunikationssysteme - Wissensvermittlung in Museen und Unternehmen
- Invention Design - Forschend-experimentelles Studium für neuartige Produkte

**Studium:**
- Grundlagenstudium (1.–3. Semester): Gestaltungspraxis, Theorie, Technik
- Schwerpunkte (4.–6. Semester): Interface Design oder Physical Computing
- Bachelor-Thesis (7. Semester)

**Bewerbung:**
- Online-Bewerbung (15.05.–15.07.)
- Hausaufgabe verpflichtend
- Arbeitsproben optional
- Eignungsprüfung am 03.06.2026
