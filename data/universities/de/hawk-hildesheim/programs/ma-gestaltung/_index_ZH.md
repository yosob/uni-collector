# 设计硕士（MA Gestaltung）— 文科硕士

## 概况

希尔德斯海姆HAWK大学设计学院的**设计硕士（Gestaltung）**是一个以研究为导向的高级设计专业，重点为**整合设计**。它建立在学院9个能力领域的基础上，强调战略思维、设计领导力和跨学科合作。

## 基本信息

| 项目 | 详情 |
|------|------|
| 学位 | 文科硕士（M.A.） |
| 学制 | 4学期（120 ECTS） |
| 入学 | 冬季和夏季学期 |
| 地点 | 希尔德斯海姆，设计学院 |
| 授课语言 | 德语 |
| 录取 | 限制录取（NC + 能力评估） |
| 作品集 | 需要（作品样本 + 选拔面试） |

## 重点方向

MA设计专业以**整合/集成设计**为核心：
- 在个性化学习路径中组合多个能力领域
- 发展设计战略和管理技能
- 参与设计导向的研究
- 领导跨学科设计项目

### 可选能力领域
本科的9个能力领域在硕士层面均可选：
广告设计、品牌设计、数字环境、色彩设计、平面设计、室内建筑、灯光设计、金属造型与珠宝、产品设计

## 课程结构

- **第1-2学期：** 高级设计项目、研究方法、整合设计概念
- **第3学期：** 毕业项目准备、专业化
- **第4学期：** 硕士论文

## 申请与录取

### 申请截止日期
- **冬季学期：** 7月15日
- **夏季学期：** 1月15日

### 选拔程序（Auswahlverfahren）
录取基于积分制（最高160分）：
| 标准 | 最高分 |
|------|--------|
| 本科毕业成绩 | 90 |
| 能力评估面试 | 60 |
| 设计职业经验 | 10 |
| **总计** | **160** |

### 入学要求
- 完成本科学位（B.A.，设计或相关领域至少180 ECTS）
- 能力评估面试（选拔面试）
- 作品样本/作品集
