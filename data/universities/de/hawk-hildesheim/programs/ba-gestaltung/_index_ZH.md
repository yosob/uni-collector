# 设计学士（BA Gestaltung）— 文学士

## 概况

希尔德斯海姆HAWK大学设计学院的**设计学士（Gestaltung）**是一个综合性设计专业，提供9个独特的能力领域。它是德国最广泛的设计专业之一，将传统手工艺与数字创新相结合。

## 基本信息

| 项目 | 详情 |
|------|------|
| 学位 | 文学士（B.A.） |
| 学制 | 6学期（180 ECTS） |
| 入学 | 冬季和夏季学期 |

| 录取 | 限制录取（NC + 能力测试） |
| 作品集 | 需要（艺术能力测试） |

## 能力领域

学生从第3学期起从**9个专业方向**中选择：

1. **广告设计** — 活动策划、文案、媒体规划
2. **品牌设计** — 品牌战略、企业识别、品牌传播
3. **数字环境** — UX/UI设计、交互媒体、网页开发
4. **色彩设计** — 色彩理论、建筑与设计中的色彩、色彩系统
5. **平面设计** — 字体排印、编辑设计、海报设计、插图
6. **室内建筑** — 空间设计、材料概念、室内规划
7. **灯光设计** — 建筑照明、灯光艺术、照明概念
8. **金属造型与珠宝** — 珠宝设计、金属加工、艺术品
9. **产品设计** — 工业设计、产品开发、原型制作

## 课程结构

- **第1-2学期：** 基础学习 — 设计基础、绘画、色彩理论、材料、数字工具
- **第3-5学期：** 在所选能力领域专业化，跨学科项目
- **第6学期：** 毕业论文项目

### 申请截止日期
- **冬季学期：** 7月15日
- **夏季学期：** 1月15日

### 艺术能力测试（künstlerische Befähigungsprüfung）
- **作品集提交期（冬季学期）：** 2月1日 – 4月30日
- **作品集提交期（夏季学期）：** 8月1日 – 10月30日

- 高等教育入学资格（高考或同等学历）
- 艺术能力测试（作品集 + 实务考试）
- 入学前实习（8周，建议）
