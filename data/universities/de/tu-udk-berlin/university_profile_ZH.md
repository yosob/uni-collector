## 设计类专业

### 硕士项目

#### Design & Computation (M.A.)
| 项目 | 详情 |
|------|------|
| **学位** | Master of Arts (M.A.) — TU Berlin + UdK Berlin 联合学位 |
| **学制** | 4学期（120 ECTS） |
| **授课语言** | 德语/英语 |
| **入学学期** | 冬季学期 |
| **专业方向** | 设计作为媒体实践、计算设计、建模与模拟、结构与系统、批判性设计思维、跨学科设计研究 |
| **作品集要求** | 需要 — 任意媒体格式（PDF、视频、音频、网站等） |
| **录取要求** | 人文/科学/工程/艺术设计/经济/法律/医学本科学位；动机信；作品集；面试 |
| **申请截止** | 4月1日 – 5月15日（冬季学期） |
| **学费** | 免费（~360.45 €/学期注册费） |
| **教学地点** | Studio Campus Charlottenburg, Straße des 17. Juni 135, 10623 Berlin |
| **链接** | [专业页面](https://www.newpractice.net/study) |

**特色**：完全跨学科的研究型项目，探索艺术、科学、社会与技术。联合学位证书含两校印章。工作室制学习，年度主题变化。

## 申请流程

1. **通过TU Berlin门户（TuPort）申请** — 提交材料 + 动机信
2. **通过Google表单上传作品集**
3. **参加面试**（6月中旬 – 8月中旬，现场）
4. **录取后**：在UdK Berlin注册，在TU Berlin登记

### 申请时间线
- **申请期**：4月1日 – 5月15日
- **面试**：6月中旬 – 8月中旬
- **开学**：10月（冬季学期）

### 作品集建议
- 接受任何媒体格式
- 展示个人方法、兴趣和技能
- 截止前定期举办作品集咨询见面会
