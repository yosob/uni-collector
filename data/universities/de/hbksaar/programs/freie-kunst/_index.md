# Freie Kunst (Fine Art / 自由艺术)

HBKsaar 的自由艺术专业提供 B.A.（8学期/240 ECTS）和 Diplom（10学期/300 ECTS）两种学位选择。专业涵盖绘画、雕塑、视频与摄影、声音艺术、概念艺术和公共艺术等方向。

## 教学特色

- **跨学科基础**：第1-2学期所有设计/艺术专业共享 Bildnerische Grundlagen 造型基础课程
- **Meisterklasse 工作室制**：第3学期起进入教授工作室，个性化指导
- **跨媒介自由**：鼓励跨媒介创作和实验
- **艺术理论与历史**：Prof. Dr. Matthias Winzen 教授主持的艺术史与艺术理论课程
- **展览实践**：定期展览和 Rundgang（校园开放日）
- **Campus Völklingen**：在联合国教科文组织世界遗产Völklinger Hütte设有项目工作坊

## 教授

- **Prof. Katharina Hinsberg** — 概念绘画
- **Prof. Gabriele Langendorf** — 绘画与素描
- **Prof. Eric Lanz** — 视频与艺术摄影
- **Prof. Andreas Oldörp** — 声音艺术（至2026年）
- **Prof. Georg Winter** — 雕塑 / 公共艺术
- **Prof. Dr. Matthias Winzen** — 艺术史与艺术理论
- **Prof. Detlef Hartung** — 光与跨媒体（代理）

## 申请

- **申请门户**: SIM-Bewerbungsportal
- **作品集**: 约20件作品，展示个人艺术表达力
- **能力测试**: Eignungsprüfung（含实践考试/面试）
- **截止日期**: 5月31日（冬季学期）
