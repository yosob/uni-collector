# 视觉传达（学士）

## 概述
视觉传达学士专业提供视觉传达设计实践与理论的深入知识。学生学习构思、设计和实现视觉信息。

## 重点领域
- 字体排印与版面设计
- 平面设计
- 插画
- 摄影
- 影像与动画
- 界面设计 / 交互设计
- 企业形象设计 / 品牌设计

## 课程结构
- **总计：** 180学分，6学期
- **专业发展领域：** 30学分
- **学士论文：** 9周完成时间
- 设计实践项目
- 视觉传达理论基础
- 工作坊实践与材料技术

## 入学要求
- 高中毕业证书（Abitur）或专业入学资格
- 含设计作品的作品集（必须提交）
- 通过能力测试
- 通过 meinStudium 门户网站在线申请

## 申请流程
1. 通过 https://studium.hbk-bs.de 在线申请，截止日期为3月15日
2. 提交数字作品集
3. 收到能力测试邀请（通过初选后）
4. 现场能力测试

## 语言要求
- 德语（教学语言）
- 国际申请者：DSH-2 或同等水平

## 联系方式
- 视觉传达研究所
- 不伦瑞克造型艺术学院
- Johannes-Selenka-Platz 1, 38118 Braunschweig
