## Design-Studiengänge

### Bachelorstudiengänge

#### Design: Produkt und Kommunikation (B.A.)
| Bereich | Details |
|---------|---------|
| **Abschluss** | Bachelor of Arts (B.A.) |
| **Dauer** | 7 Semester (210 ECTS) |
| **Sprache** | Deutsch |
| **Beginn** | Wintersemester |
| **Fakultät** | Fakultät Design |
| **Schwerpunkte** | Produktdesign, Kommunikationsdesign, Interfacedesign, Interaktives Systemdesign, Integratives Entwerfen |
| **Portfolio** | Erforderlich — 15-20 Arbeiten, physische Mappe |
| **Eignungsprüfung** | Mappe (40%) + Praktische Prüfung (40%) + Kolloquium (20%) |
| **Bewerbungsfrist** | 15. Juli (Wintersemester) |
| **Gebühren** | Keine Studiengebühren (~350 € Semesterbeitrag) |
| **Link** | [Studiengangsseite](https://www.htw-dresden.de/studium/vor-dem-studium/studienangebot/bachelor-produkt-und-kommunikationsdesign) |

**Besonderheiten**: Integratives Designkonzept — Physisches, Medien- und Kommunikationsdesign werden von Anfang an zusammen unterrichtet. 16 Wochen Praktikum im 6. Semester.

### Masterstudiengänge

#### Design: Products and Interactions (M.A.)
| Bereich | Details |
|---------|---------|
| **Abschluss** | Master of Arts (M.A.) |
| **Dauer** | 3 Semester (90 ECTS) |
| **Sprache** | Deutsch |
| **Beginn** | Nur Sommersemester |
| **Fakultät** | Fakultät Design |
| **Schwerpunkte** | Produktdesign, Interfacedesign, Interaktives Systemdesign, Design Research, User Experience, Hybride Produkte |
| **Portfolio** | Erforderlich — PDF-Portfolio |
| **Zulassung** | Bachelor in Design (210 ECTS), Portfolio, Motivationsschreiben, Interview |
| **Bewerbungsfrist** | 15. Januar (Sommersemester) |
| **Gebühren** | Keine Studiengebühren (~350 € semesterbeitrag) |
| **Link** | [Studiengangsseite](https://www.htw-dresden.de/studium/vor-dem-studium/studienangebot/master-design-products-and-interactions) |

**Besonderheiten**: Konsekutiver Master, aufbauend auf den HTW-Bachelor. Fokus auf hybride Produkte und Anwendungen an der Schnittstelle von Digitalität und Physis.

## Bewerbungsverfahren

1. **Onlinebewerbung** über das HTW Dresden Bewerbungssystem
2. **Einreichung der Mappe** (Portfolio)
3. **Teilnahme an der Eignungsprüfung** (Praktische Prüfung + Kolloquium)
4. **Zulassungsbescheid** abwarten

### Wichtige Termine
- **Bachelor (WS)**: Bewerbung 15. Mai – 15. Juli
- **Master (SS)**: Bewerbung bis 15. Januar
