## 学校特色

德累斯顿工程与经济应用技术大学是萨克森州最大的应用技术大学之一，位于历史文化名城德累斯顿。学校设有8个学院，以实践导向的教学和紧密的企业合作著称。**Fakultät Design（设计学院）** 是其特色学院之一，以独特的**整合性设计教育**理念闻名。

### 8个学院
1. Informatik / Sprachen（计算机科学与语言）
2. Elektrotechnik（电气工程）
3. Bauingenieurwesen（土木工程）
4. Maschineningenieurwesen（机械工程）
5. Landbau / Umwelt / Chemie（农业/环境/化学）
6. **Design（设计）** ⭐
7. Wirtschaftswissenschaften（经济学）
8. Geoinformation（地理信息）

## 设计学院 (Fakultät Design)

### 教学理念

Fakultät Design 的核心理念是**整合性设计**（Integratives Design）：从第一学期开始就将产品设计（Produktdesign）、传达设计（Kommunikationsdesign）和交互设计（Interfacedesign）三个维度融合教学，而非作为独立方向分科。

学院的座右铭是：**"混合产品与应用的整合性设计"**（Integratives Entwerfen hybrider Produkte und Anwendungen），即物理产品、软件产品和网络服务的整体设计。

### 规模
- **学生**: 约150人
- **教授**: 9位（+1位荣誉教授）
- **师生比**: 约16:1，提供密集个性化指导

### 教学特点
- 小班教学，密集指导
- 项目制学习
- 每学期**Werkschau**（公开作品展览）
- 与企业的合作项目
- 完善的工坊与实验室设施

## 设计类专业

### 本科

#### Design: Produkt und Kommunikation B.A.
| 项目 | 内容 |
|------|------|
| 学位 | Bachelor of Arts (B.A.) |
| 学制 | 7学期 (210 ECTS) |
| 入学 | 仅冬季学期（10月） |
| 语言 | 德语 |
| 录取 | 开放招生 + 能力测试 (Eignungsprüfung) |
| 实习 | 第6学期，16周 |
| 特色 | 整合性设计教育；产品、传达与交互三位一体 |

**课程结构**:
- 第1-3学期: 设计基础（二维/三维/交互）、绘图、色彩、排版、材料工艺
- 第4-5学期: 专业深化（产品/传达/交互设计项目）
- 第6学期: 专业实习（16周）
- 第7学期: 毕业设计 + 展览

### 本科申请流程（Eignungsprüfung）
1. 在线申请
2. 提交作品集 (Mappe) — 评分权重40%
3. 实践考试 (Praktische Prüfung) — 评分权重40%
4. 面试 (Kolloquium, 约15-20分钟) — 评分权重20%

### 硕士申请材料
1. 作品集 (PDF格式)
2. 动机信 (Motivationsschreiben)
3. 本科毕业证书和成绩单
4. 简历 (Lebenslauf)
