## Design Programs

### Bachelor Programs

#### Design: Product and Communication (B.A.)
| Item | Details |
|------|---------|
| **Degree** | Bachelor of Arts (B.A.) |
| **Duration** | 7 semesters (210 ECTS) |
| **Language** | German |
| **Start** | Winter semester |
| **Department** | Fakultät Design |
| **Focus Areas** | Product Design, Communication Design, Interface Design, Interactive System Design, Integrative Design |
| **Portfolio Required** | Yes — 15-20 works, physical portfolio |
| **Admission Test** | Portfolio (40%) + Practical exam (40%) + Interview (20%) |
| **Application Deadline** | July 15 (winter semester) |
| **Tuition** | Free (~€350/semester fee) |
| **Link** | [Program page](https://www.htw-dresden.de/studium/vor-dem-studium/studienangebot/bachelor-produkt-und-kommunikationsdesign) |

**Highlights**: Integrative design philosophy combining physical, media, and communication design from day one. 16-week internship in semester 6.

### Master Programs

#### Design: Products and Interactions (M.A.)
| Item | Details |
|------|---------|
| **Degree** | Master of Arts (M.A.) |
| **Duration** | 3 semesters (90 ECTS) |
| **Language** | German |
| **Start** | Summer semester only |
| **Department** | Fakultät Design |
| **Focus Areas** | Product Design, Interface Design, Interactive System Design, Design Research, User Experience, Hybrid Products |
| **Portfolio Required** | Yes — PDF portfolio |
| **Admission Requirements** | Bachelor in design (210 ECTS), portfolio, motivation letter, interview |
| **Application Deadline** | January 15 (summer semester) |
| **Tuition** | Free (~€350/semester fee) |
| **Link** | [Program page](https://www.htw-dresden.de/studium/vor-dem-studium/studienangebot/master-design-products-and-interactions) |

**Highlights**: Consecutive master building on HTW bachelor. Focus on hybrid products and applications at the intersection of digital and physical.

## How to Apply

1. **Online application** via HTW Dresden application system
2. **Submit portfolio** (Mappe)
3. **Attend aptitude test** (practical exam + interview)
4. **Receive admission notification**

### Key Dates
- **Bachelor (WS)**: Apply May 15 – July 15
- **Master (SS)**: Apply by January 15
