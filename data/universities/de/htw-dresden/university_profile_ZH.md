## 设计类专业

### 本科项目

#### Design: Produkt und Kommunikation (B.A.)
| 项目 | 详情 |
|------|------|
| **学位** | Bachelor of Arts (B.A.) |
| **学制** | 7学期（210 ECTS） |
| **授课语言** | 德语 |
| **入学学期** | 冬季学期 |
| **所属学院** | Fakultät Design |
| **专业方向** | 产品设计、传达设计、界面设计、交互系统设计、整合性设计 |
| **作品集要求** | 需要 — 15-20件作品，实体作品集 |
| **入学考试** | 作品集(40%) + 实践考试(40%) + 面试(20%) |
| **申请截止** | 7月15日（冬季学期） |
| **学费** | 免费（~350 €/学期注册费） |
| **链接** | [专业页面](https://www.htw-dresden.de/studium/vor-dem-studium/studienangebot/bachelor-produkt-und-kommunikationsdesign) |

**特色**：整合性设计理念，从第一天起融合物理、媒体和传达设计。第6学期16周实习。

### 硕士项目

#### Design: Products and Interactions (M.A.)
| 项目 | 详情 |
|------|------|
| **学位** | Master of Arts (M.A.) |
| **学制** | 3学期（90 ECTS） |
| **授课语言** | 德语 |
| **入学学期** | 仅夏季学期 |
| **所属学院** | Fakultät Design |
| **专业方向** | 产品设计、界面设计、交互系统设计、设计研究、用户体验、混合产品 |
| **作品集要求** | 需要 — PDF格式作品集 |
| **录取要求** | 设计本科（210 ECTS）、作品集、动机信、面试 |
| **申请截止** | 1月15日（夏季学期） |
| **学费** | 免费（~350 €/学期注册费） |
| **链接** | [专业页面](https://www.htw-dresden.de/studium/vor-dem-studium/studienangebot/master-design-products-and-interactions) |

**特色**：连续硕士，衔接HTW本科。聚焦数字与物理交汇处的混合产品与应用。

## 申请流程

1. **在线申请** — 通过HTW Dresden申请系统
2. **提交作品集**（Mappe）
3. **参加能力测试**（实践考试 + 面试）
4. **等待录取通知**

### 关键日期
- **本科（冬季学期）**：5月15日 – 7月15日申请
- **硕士（夏季学期）**：1月15日前申请
