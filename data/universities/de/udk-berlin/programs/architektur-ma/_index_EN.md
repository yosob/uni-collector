# Architecture (Master of Arts)

## Program Overview

Architecture Master at UdK Berlin — artistic-academic specialization in architecture with individual focus. 4 semesters, 120 CP.

## Application Process

1. **Online Registration**: Create an account at my.udk-berlin.de
2. **Pay Application Fee**: EUR 40.00
3. **Submit Application Documents**: Portfolio, bachelor's certificate, transcripts, etc.
4. **Pre-selection**: Based on application documents
5. **Admission Exam**: Interview / portfolio presentation

- **German**: C1 (B2 certificate accepted at time of application)
