# 战略产品与创新发展 文学硕士

## 专业概况

战略产品与创新发展（Strategische Produkt- und Innovationsentwicklung）硕士专业是一个4学期的跨学科项目，结合设计、工程和商业视角，为毕业生在产品创新领域的领导角色做好准备。

## 关键信息

| 详情 | 数值 |
|------|------|
| 学位 | 文学硕士（M.A.） |
| 学制 | 4学期（120 ECTS） |
| 入学 | 冬季+夏季学期 |
| 语言 | 德语+英语 |
| 录取 | 开放录取（无录取人数限制） |
| 每学期ECTS | 30 |

## 入学要求

- 以下领域的学士学位（180 ECTS）：
  - 工业设计/产品设计
  - 机械工程
  - 工商管理
  - 或相关领域
- 最低成绩：2.0或以上
- 无录取人数限制（开放录取）

## 重点领域

- 战略产品开发
- 创新管理与方法论
- 设计思维与过程
- 产品战略
- 可持续产品创新
- 跨学科合作

## 就业前景

毕业生就业于：
- 产品开发管理
- 创新管理
- 设计战略咨询
- 研发部门
- 创业领域

## 院系

隶属于伍珀塔尔大学设计与艺术学院内的工业设计系。
