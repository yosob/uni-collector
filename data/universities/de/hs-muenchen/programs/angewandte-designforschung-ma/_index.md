# MA Angewandte Designforschung — Hochschule München

## 专业概述

慕尼黑应用技术大学的应用设计研究硕士专业（MA Angewandte Designforschung）是一个以研究为导向的硕士项目，重点聚焦于"社会变迁与转型过程"（Sozialer Wandel und transformative Prozesse）以及"设计理论与设计文化"（Designtheorie und Designkultur）两大领域。

学生在项目中以研究-实验的方式，探讨社会设计（Social Design）和转型设计（Transformationsdesign）课题，关注重大的社会、技术或文化变革。该项目不仅关注人工制品（Artefakte）的开发，更注重心智产物（Mentefakte）和社会产物（Soziofakte）的创造。

## 研究重点

### 社会变迁与转型过程
- 社会设计（Social Design）
- 转型设计（Transformationsdesign）
- 数字化社会变迁
- 资源保护与可持续发展

### 设计理论与设计文化
- 设计科学（Designwissenschaft）
- 设计理论（Designtheorie）
- 设计批评（Designkritik）
- 日常与消费文化现象研究

## 申请信息

### 申请时间线 ⭐
- **入学时间**: 每年**夏季学期**（3月中旬）
- **申请期**: **11月15日 — 1月15日**
- ⚠️ 注意：与其他多数设计专业不同，本专业在夏季学期入学

### 入学条件
- 设计相关本科或文凭学位，成绩"良好"（2.5）及以上
- 至少 210 ECTS 学分，含6个理论学期 + 1个实践学期（或至少18周相关职业经验）
- 如只有 180 ECTS 且无实践学期，可在第一学年补齐
- 国际申请者：德语 DSH-2 级或 TestDaF 3 级

### 申请流程
1. **PRIMUSS 在线申请**: 在申请期内提交数据与文件
2. **HMster 作品集上传**: 在 nine.hm.edu 注册并上传数字作品集
3. **录取面试**: 与 MA 考试委员会（3位教授）进行面试

### 作品集要求
| 内容 | 要求 |
|------|------|
| 动机信 | 必须提交，通过表单输入 |
| 简历与照片 | 必须提交（照片可选） |
| 作品样本 | 最多25件 |
| 文件格式 | .png, .jpg, .gif, .pdf |
| 文件大小 | 每个文件最大 50 MB |
| 附加说明 | 可为每件作品添加说明文字 |

⚠️ **重要**: PRIMUSS 申请和 HMster 作品集上传两者都须完成。建议提前数天上传作品集以避免服务器拥挤。

## 课程特色

- **研究-实验方法**: 以研究性实验方式探索设计问题
- **跨学科与跨文化**: 面向国内外设计学院毕业生及相关专业背景学生
- **设计研究实践**: 实践导向的设计研究（praxisbasierte Designforschung）
- **应用设计理论**: 理论与实践结合（anwendungsbezogene Designtheorie）
- **与 dci 研究所紧密联系**: 可参与 Designkulturen Institut 的研究项目
