# 艺术管理与文化工作 B.A.

## 专业概述

波茨坦应用科学大学的艺术管理与文化工作本科项目是一个综合型项目，将文化、媒体和社会科学议题与企业管理和管理理论相连接。为学生从事文化工作和文化管理的多元职业领域做准备。

## 学习结构

| 学期 | 重点 |
|------|------|
| 1–2 | 基础：文化学、媒体学、社会科学 |
| 3–4 | 管理基础：企业管理、营销、项目管理 |
| 5–6 | 专业化和实践项目 |
| 7–8 | 高级模块和学士论文 |

## 专业特色

- **综合型培养**：文化、媒体和管理的广泛教育
- **跨学科**：融合人文学科、社会科学和商业
- **高实践比例**：以真实文化项目为基础的项目式学习
- **广泛的就业领域**：文化机构、节庆活动、媒体、教育
- **仅冬季学期**：仅在冬季学期招生

## 重点方向

- 文化与人文学科基础
- 媒体学与媒体实践
- 社会科学方法
- 企业管理
- 文化管理与营销
- 项目管理
- 文化教育与推广
- 实践项目工作

## 入学要求

- 普通高等教育入学资格（Abitur）
- 有文化/实践经验者优先
- 通过FH Potsdam门户网站申请

## 就业方向

毕业生工作领域：
- 文化管理与行政
- 节庆与活动组织
- 文化教育与推广
- 文化政策与咨询
- 媒体与公共关系
- 文化机构（剧院、博物馆、画廊）
- 社区文化中心
