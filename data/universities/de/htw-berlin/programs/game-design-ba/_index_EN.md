# Game Design (BA)

## Overview

The Bachelor's program in Game Design at HTW Berlin provides interdisciplinary competencies in game development, game art, game design and interactive media. The program is characterized by a strong project focus and close integration with the DE:HIVE Games Incubator.

## Curriculum

- **Game Design**: Game mechanics, level design, system design, game thinking
- **Game Art**: Character design, environment design, production design
- **Game Development**: Prototyping, programming, production
- **Interactive Media**: Narrative systems, user experience
- **Project Work**: Regular game projects in teams

## Specialization

In the later stages of the program, students can specialize in various areas:
- Game Design
- Game Art
- Game Programming
- Producing/Management

## Highlights

- **DE:HIVE**: Professional studio environment and incubator for game projects
- Regular awards at national and international competitions
- Close community with Discord and Matrix channels
