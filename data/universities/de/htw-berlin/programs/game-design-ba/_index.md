# Game Design (BA)

## Übersicht

Der Bachelorstudiengang Game Design an der HTW Berlin vermittelt interdisziplinäre Kompetenzen in Spieleentwicklung, Game Art, Game Design und interaktiven Medien. Das Studium zeichnet sich durch einen starken Projektbezug und die enge Anbindung an den DE:HIVE Games Incubator aus.

## Studieninhalte

- **Game Design**: Spielmechaniken, Level Design, System Design, Game Thinking
- **Game Art**: Character Design, Environment Design, Production Design
- **Spieleentwicklung**: Prototyping, Programmierung, Produktion
- **Interaktive Medien**: Narrative Systeme, User Experience
- **Projektarbeit**: Reguläre Spieleprojekte in Teams

## Spezialisierung

Im späteren Studienverlauf können Studierende sich in verschiedenen Bereichen spezialisieren:
- Game Design
- Game Art
- Game Programming
- Producing/Management

## Besonderheiten

- **DE:HIVE**: Professionelle Studioulmgebung und Incubator für Spieleprojekte
- Regelmäßige Auszeichnungen bei nationalen und internationalen Wettbewerben
- Enge Community mit Discord und Matrix Kanälen
