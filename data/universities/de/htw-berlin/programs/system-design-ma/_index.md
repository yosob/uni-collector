# System Design / Game Design (MA)

## Übersicht

Der Masterstudiengang System Design / Game Design an der HTW Berlin fokussiert auf Spiele-Entwicklung, Playable Environments, Game Thinking und eXtended Realities. Das Programm ermöglicht künstlerische Vorhaben, Forschungsprojekte oder Gründungen im Team.

## DE:HIVE — Studioulmgebung

Masterstudierende arbeiten im **DE:HIVE**, einer professionellen und multifunktionalen Studioulmgebung mit erstklassiger technischer Infrastruktur in der Slabystraße in Berlin-Schöneweide.

## Bekannte Alumni-Projekte

- **Dorfromantik** (Toukana) — Minimalistic Landscaping Puzzle Strategy
- **AnotherWhere** — Level- und Gameplay-Generator
- **Emberstorm** — Aufbau eines Indiestudios
