## URL 格式说明

- **本科课程**: `https://www.lancaster.ac.uk/study/undergraduate/courses/{slug}/{year}/`
- **研究生课程**: `https://www.lancaster.ac.uk/study/postgraduate/postgraduate-courses/{slug}/{year}/`
- **当前可用年份**: 2026, 2027（种子 URL 使用 2024）
- **slug 规则**: `{name}-{degree}-hons-{ucas-code}`（全小写，连字符分隔）
- **变体命名**: 标准（3 年）→ `{name}-ba-hons-{code}`；实习年 → `{name}-placement-year-ba-hons-{code}`；海外学习 → `{name}-study-abroad-ba-hons-{code}`
- **BSc 变体**: 实习年用 `industry` 而非 `placement-year`，如 `marketing-and-design-industry-bsc-hons-nw53`
- **课程页锚点**: `#overview`, `#careers`, `#course-entry`, `#structure`, `#fees`

> 以下列出每个专业的标准（3 年制）变体为主要 URL。每个专业通常还有 Placement Year（+1 年）和 Study Abroad（+1 年）变体，UCAS code 不同但页面结构一致。
