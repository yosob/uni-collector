## 概述

兰卡斯特大学的美术与电影是一个令人兴奋的学位，将创意实践与批判性电影研究相结合。该课程将基于工作室的美术创作与电影分析、理论和制作融合在一起，让你能够深入探索这两个学科并发展自己的创意声音。

你将从兰卡斯特当代艺术学院（LICA）丰富的跨学科环境中受益，这里汇聚了艺术家、电影制作人、设计师和表演者。该课程鼓励你在发展绘画、雕塑、电影制作等实践技能的同时，批判性地思考艺术和电影。

在整个学位课程中，你将可以使用专用工作室、电影剪辑套件、专业设备和技术支持。你将培养细致分析、创意思维、视觉传达和团队协作能力——这些都是雇主高度重视的技能。

## 为什么在兰卡斯特学习美术与电影？

- **跨学科创意融合**：将你对视觉艺术的热情与对全球电影的深入理解相结合
- **行业标准设施**：使用专用工作室、暗房、电影剪辑套件和专业设备
- **专家教学**：在研究密集型大学中向从业艺术家和顶尖电影学者学习
- **灵活的课程**：从广泛的选修模块中进行选择，根据你的兴趣定制学位
- **专业实践**：在最后一年开发创意作品集并完成毕业论文项目
- **职业准备**：培养视觉分析、批判性思维、沟通和项目管理等可转移技能

## 课程结构

该课程为期三年。每年你将修读核心模块和选修模块的组合，在建立坚实基础的同时发展自己的兴趣。

### 第一年

**核心模块：**

| 模块 | 说明 |
|------|------|
| 电影研究导论 | 介绍电影理论、分析和历史的关键方法，同时融入实践性电影制作体验。你将通过分析好莱坞和全球传统的电影案例来研究摄影、剪辑、声音和表演。 |
| 艺术史与语境 | 为你提供扎实的艺术史基础，从现代主义时期到当代。从欧洲现代主义的兴起开始，你将追踪其全球影响，并思考这些语境与你自身实践的相关性。 |

**选修模块：**

| 模块 | 说明 |
|------|------|
| 创意关注 | 一个变革性的模块，介绍本科阶段的美术学习。重点关注艺术家如何通过观察、倾听、触摸、手势和标记来探索世界，你将探索多种多样的媒介。 |
| 理解媒体与文化 | 介绍媒体与文化研究中的主要辩论、理论家和思想家。你将接触到来自电视、电影、新闻和数字媒体的多种材料。 |

### 第二年

**核心模块：**

| 模块 | 说明 |
|------|------|
| 全球电影 | 通过研究不同历史和地区背景的电影，探索全球电影的风格、工业和文化多样性。你将考察电影风格和形式的变化，同时发展分析能力。 |

**选修模块：**

| 模块 | 说明 |
|------|------|
| 艺术理论与语境 | 在理论理解与艺术实践的关系基础上，介绍与当代艺术实践和思想相关的广泛理论和观念，包括新唯物主义、去殖民化和身份政治。 |
| 创意转化 | 想象力、实验、创意冒险和转化是该模块的核心。你将在鼓励跨多种媒介和过程进行实验和冒险的工作坊中进行创意实践。 |
| 电影理论与美学 | 超越入门概念，深入探索电影理论，考察电影如何通过形式、表征和观众参与来构建意义。你将批判性地评估电影与艺术表达和观众关系。 |
| 短片制作 | 以协作方式开发、制作和完成一部短篇剧情电影，同时采用可持续的行业标准制作实践。重点在于技术能力、创意问题解决和反思性实践。 |

### 第三年

**核心模块：**

| 模块 | 说明 |
|------|------|
| 独立项目（电影、英语、创意写作与媒体） | 你学业的顶点，结合你所学的所有知识，就你选择的主题进行深入的自导研究项目。你将提出研究领域，并与合适的导师合作。 |
| 高级艺术实践 | 完善你的实践技能，开发并展示适合美术学位展的专业实践作品。你作为艺术家的成长将通过讲座、工作坊、辅导和小组评论得到支持。 |

**选修模块：**

| 模块 | 说明 |
|------|------|
| 全球电影类型 | 在全球背景下考察电影类型的演变，重点关注当代电影人如何在既定和经典类型传统中进行创新，包括科幻、恐怖和喜剧。 |
| 纪录片制作 | 一个高级的实践模块，提供行业导向的纪录片制作方法。以小组形式工作，你将研究、开发和制作一部短篇纪录片。 |
| 实验电影制作 | 一个动手实践的模块，探索实验电影制作及其如何挑战主流叙事，通过批判性讨论和实践工作坊打破传统的观影模式。 |

## 就业前景

美术与电影专业的毕业生培养了一系列受到各行业雇主重视的可转移技能。潜在的职业路径包括：

- **创意产业**：艺术家、电影制作人、策展人、艺术总监
- **媒体与传播**：电影制作、视频编辑、内容创作
- **教育领域**：教学、讲学、美术馆教育
- **文化领域**：博物馆和美术馆工作、艺术管理、文化遗产
- **继续深造**：攻读美术、电影研究、策展实践或相关领域的研究生学位

创意实践与批判性思维的结合为你准备了需要视觉素养和分析能力的职位。许多毕业生也成为独立的艺术家和电影制作人。
