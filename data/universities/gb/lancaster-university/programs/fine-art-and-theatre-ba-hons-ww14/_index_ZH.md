## 概述

兰卡斯特大学的美术与戏剧提供了一个独特的机会，探索和发展美术与戏剧之间丰富的对话。该联合学位设在兰卡斯特当代艺术学院（LICA），将基于工作室的美术实践与戏剧研究和创作结合在一起，让你能够在两个学科之间进行创造性的跨界工作。

该课程鼓励你批判性地思考视觉艺术与表演之间的关系，培养你作为艺术家和戏剧实践者的双重技能。在专业学者和执业艺术家的支持下，你将获得跨媒介创作的自由——从绘画、雕塑和装置到编创表演、身体剧场和应用戏剧。

在整个学位课程中，你将使用专用工作室、表演空间、工作坊和技术支持。你将培养创意思维、视觉传达、表演创作和团队协作等技能，为你在创意和文化产业中的广泛职业发展做好准备。

## 为什么在兰卡斯特学习美术与戏剧？

- **跨学科创意融合**：将你对视觉艺术的热情与戏剧创作相结合，在鼓励两门学科对话的课程中发展独特技能
- **行业标准设施**：使用专用工作室、表演空间、工作坊和专业设备
- **专家教学**：在研究密集型大学中向执业艺术家、戏剧创作者和顶尖学者学习
- **灵活的课程**：从广泛的选修模块中进行选择，根据你的创意兴趣定制学位
- **专业实践**：在最后一年开发创意作品集并完成重大表演项目
- **职业准备**：培养视觉分析、表演、批判性思维、沟通和项目管理等可转移技能

## 课程结构

该课程为期三年。每年你将修读核心模块和选修模块的组合，在建立坚实基础的同时发展自己的创意兴趣。

### 第一年

**核心模块：**

| 模块 | 说明 |
|------|------|
| 理解文化 | 介绍文化研究以及文化如何塑造并受艺术塑造的方式。你将探索构成美术与戏剧研究基础的关键概念和批判方法。 |
| 创意关注 | 一个变革性的模块，介绍本科阶段的美术学习。重点关注艺术家如何通过观察、倾听、触摸、手势和标记来探索世界，你将探索多种多样的媒介。 |

**选修模块：**

| 模块 | 说明 |
|------|------|
| 写作实践 | 探索写作与创意实践之间的关系，培养你在艺术和戏剧语境中将文本作为创意和批判工具使用的能力。 |
| 创意转化 | 想象力、实验、创意冒险和转化是该模块的核心。你将在鼓励跨多种媒介和过程进行实验和冒险的工作坊中进行创意实践。 |

### 第二年

**核心模块：**

| 模块 | 说明 |
|------|------|
| 艺术理论与语境 | 在理论理解与艺术实践的关系基础上，介绍与当代艺术实践和思想相关的广泛理论和观念，包括新唯物主义、去殖民化和身份政治。 |
| 戏剧创作2：中级表演项目 | 通过中级表演项目发展你的戏剧创作技能。你将探索多种创作表演作品的方法，培养你作为协作戏剧创作者的能力。 |

**选修模块：**

| 模块 | 说明 |
|------|------|
| 表演构图 | 探索表演构图的原理，考察空间、时间、动作和材料如何被组织以创造引人入胜的戏剧体验。 |
| 应用戏剧 | 研究戏剧和表演如何应用于社区、教育和社会语境，发展你对参与式和社会参与型表演实践的理解。 |
| 艺术写作 | 在多种艺术语境中发展你的写作技能，包括批评写作、评论、艺术家陈述和创意文本。 |

### 第三年

**核心模块：**

| 模块 | 说明 |
|------|------|
| 高级艺术实践 | 完善你的实践技能，开发并展示适合美术学位展的专业实践作品。你作为艺术家的成长将通过讲座、工作坊、辅导和小组评论得到支持。 |
| 戏剧创作3：高级表演项目 | 你戏剧学习的顶点，你将构思、开发和呈现一个高级表演项目，运用在整个学位期间发展的技能和知识。 |

**选修模块：**

| 模块 | 说明 |
|------|------|
| 艺术写作 | 在多种艺术语境中发展你的写作技能，包括批评写作、评论、艺术家陈述和创意文本。 |
| 现场表演批评 | 通过分析和讨论现场表演发展你的批判声音，培养关于当代戏剧和表演作品的写作和评估技能。 |
| 未来节 | 一个沉浸式模块，探索节庆在当代文化中的作用，最终完成一个面向公众的节庆活动的规划和实施。 |

## 就业前景

美术与戏剧专业的毕业生培养了一系列受到各行业雇主重视的可转移技能。潜在的职业路径包括：

- **创意产业**：艺术家、戏剧创作者、表演者、导演、策展人
- **艺术管理**：节庆管理、艺术机构、文化项目策划
- **教育领域**：教学、讲学、美术馆和剧院教育
- **文化领域**：博物馆和美术馆工作、艺术政策、社区艺术
- **表演与媒体**：表演、导演、舞台管理、制作
- **继续深造**：攻读美术、戏剧、表演研究或相关领域的研究生学位

创意实践与批判性思维的结合为你准备了需要视觉素养和协作表演能力的职位。许多毕业生也成为独立的艺术家和戏剧实践者。
