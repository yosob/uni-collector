## Overview

The Computer Game Development MSc at Kingston University provides a comprehensive education in the technical and creative aspects of game development. The course is designed for graduates who want to enter the games industry or advance their existing skills in game programming, design and production.

You will learn to design, develop and implement games using industry-standard game engines and tools. The course covers game programming, game design principles, artificial intelligence for games, graphics programming and the game development pipeline.

## Course Structure

### Semester 1

| Module | Credits |
|--------|---------|
| Game Programming | 30 |
| Game Design and Gameplay | 30 |

**Game Programming** — Develops your programming skills in the context of game development, covering game engines (Unity, Unreal), game architecture, physics simulation and real-time rendering.

**Game Design and Gameplay** — Explores the principles of game design, including game mechanics, level design, player experience, narrative design and game balance.

### Semester 2

| Module | Credits |
|--------|---------|
| Artificial Intelligence for Games | 30 |
| Game Development Group Project | 30 |

**Artificial Intelligence for Games** — Covers AI techniques used in games, including pathfinding, decision-making systems, procedural generation and machine learning applications in game development.

**Game Development Group Project** — A collaborative module in which you work in a team to develop a complete game from concept to playable prototype, simulating industry production processes.

### Semester 3 (Major Project)

| Module | Credits |
|--------|---------|
| Major Project | 60 |

**Major Project** — An individual project in which you develop a significant game or game-related technology project. You will demonstrate mastery of technical and creative game development skills.
