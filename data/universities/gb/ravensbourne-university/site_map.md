## Program Page Template

Each program page follows a consistent structure with these sections:

1. **Hero** — Course title, UCAS code, duration, campus
2. **Overview** — Course introduction and key features
3. **Key Facts** — UCAS code, duration, fees, start date, campus location, award
4. **Modules** — Year-by-year curriculum breakdown
5. **Entry Requirements** — Qualifications, tariff points, English language requirements
6. **Teaching and Assessment** — Teaching methods, assessment types
7. **Graduate Outcomes** — Career paths and employability
8. **Portfolio Requirements** — For creative programs, portfolio/showreel guidance
9. **Apply CTA** — Link to UCAS / direct application
10. **Related Programs** — Links to similar courses

### URL Patterns

- **Animation & Games programs** use short URLs: `/{slug}` (e.g., `/ba-hons-animation`)
- **Other undergraduate programs** use: `/study/undergraduate/{slug}`
- **All postgraduate programs** use: `/study/postgraduate/{slug}`
- **Programs with "with Industry Year"** variants have separate pages
- **Integrated Foundation Year** is an option on the same program page (not a separate URL)
