## BDes (Hons) Animation

**BDes（荣誉）动画设计学士** 是邓迪大学 Duncan of Jordanstone 艺术与设计学院（DJCAD）旗下的本科课程。该课程以叙事为核心，聚焦角色动画与故事讲述，培养学生掌握 2D 和 3D 动画技术。

### 课程特色

- 🎬 **叙事驱动**：以故事讲述和角色塑造为核心教学理念
- 🖊️ **双领域覆盖**：同时教授 2D 传统动画和 3D 数字动画技术
- 🏭 **行业联系**：与 BBC、迪士尼、Rockstar Games 等知名企业保持合作关系
- 📐 **基础年制**：第一年为 Art & Design 通识基础年，第二年起专攻动画方向
- 🎓 **实践导向**：通过工作室实践、行业真实项目和个人创作积累专业经验

### 入学要求

| 项目 | 要求 |
|------|------|
| 苏格兰 Highers | BBBC（含英语或相关创意科目） |
| A-Levels | BCC |
| UCAS 积分 | 104 分以上 |
| 雅思 | 总分 6.0（单项不低于 5.5） |
| 作品集 | **必须提交**数字作品集 |
| 申请方式 | 通过 UCAS 申请（课程代码 W615） |

### 学费（2025/26 学年）

| 学生类别 | 年费 |
|----------|------|
| 苏格兰/欧盟学生 | 由 SAAS 资助，免学费 |
| 英国其他地区 | £9,250 |
| 国际学生 | £21,900 |

### 就业方向

毕业生可从事动画师、分镜师、角色设计师、动态图形设计师、概念艺术家、3D 建模师、视觉特效师、艺术总监等职业，广泛活跃于影视、游戏、广告及数字媒体行业。
