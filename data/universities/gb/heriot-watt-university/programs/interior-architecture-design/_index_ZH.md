# BA (Hons) 室内建筑与设计

## 课程概述

我们的 BA (Hons) Interior Architecture and Design 课程提供**创意设计技能的综合训练**，为你进入行业做好准备。你将运用概念性方法，富有想象力地改造或重新利用室内环境。核心领域包括**可持续性**、**以人为本的设计**、手工和数字制作技能、材料、绘图、家具与细部设计、技术与数字传达。

你将学习创新新技术，将室内设计想法变为现实，与 **Adidas、Herman Miller** 和 **Accor Hotel Group** 等品牌合作真实项目。

课程在**苏格兰边境校区**授课，**Go Global** 国际交流项目提供前往迪拜校区或全球合作院校学习的机会。

## 为什么选择这个课程？

- **苏格兰室内设计排名第3**（卫报大学指南 2026）
- 注册建筑师和研究型学者担任教学
- 与 **Adidas、Herman Miller、Accor Hotel Group** 的真实行业项目
- **Go Global** 国际交流项目——在全球合作院校学习
- 毕业生就职于 **AECOM、Area、IKEA**
- 第三年可选行业实习和实战项目

## 入学要求

- **学术要求（大一标准）：** 苏格兰高中 BBBB / A-Level BCC + 数字作品集 / IB 29分 / BTEC Extended Diploma DMM
- **最低要求（公平入学）：** 苏格兰高中 BBBC / A-Level BCC / IB 29分
- **大二直入：** Advanced Highers BB / A-Level BBC + 数字作品集 / IB 30分
- **大三直入：** HND BB级相关科目（3D设计、室内设计）
- **英语要求：** 雅思总分 6.0，各单项不低于 5.5
- **作品集：** 须提交数字作品集（10-20张图片，展示研究、设计发展、技术技能、视觉传达）

## 课程结构

### 第一学年——基础（公共课程）

所有时尚、传播、营销与零售方向学生的公共基础学年，培养分析、评估和语境化能力。

| 学期 | 课程 |
|------|------|
| 9月 | 设计语境与过程 |
| 9月 | 设计传达 |
| 1月 | 设计原型制作 |
| 1月 | 设计挑战 |

### 第二学年——专业化方向

发展室内建筑基础，包括构造、住宅设计和工作空间设计。

**必修课程：**

| 学期 | 课程 |
|------|------|
| 9月 | 设计思维 |
| 9月 | 室内建筑构造 |
| 9月 | 设计工作室1——住宅设计 |
| 1月 | 设计研究方法 |
| 1月 | 设计工作室2——工作空间设计 |

**选修课程（1月——选择一门）：**

- 艺术化智能设计
- 设计模拟与建模
- 趋势预测与媒体内容制作
- 产品创新
- 设计中的循环经济

### 第三学年——专业实践

通过竞赛工作室、服务行业设计项目和专业实践与行业对接。

**必修课程：**

| 学期 | 课程 |
|------|------|
| 9月 | 设计竞赛工作室 |
| 9月 | 设计工作室3——服务行业设计 |
| 9月 | 专业实践 |
| 1月 | 设计研究规划 |

**选修课程（1月）：**

- 行业实习
- 实战项目

### 第四学年——荣誉学年

通过荣誉项目和最终展示巩固你的技能。

| 时间段 | 课程 |
|--------|------|
| 9月 | 专业创意未来 |
| 1月 | 设计展示 |
| 全学年 | 荣誉项目 |
| 全学年（选修） | 营销管理与战略 |
| 全学年（选修） | 技术与创新 |

## 就业前景

毕业生在室内和空间设计领域从事多元化职业：

- **室内设计师**——住宅、商业、酒店
- **空间设计师 / 建筑顾问**
- **设计经理 / 创意总监**
- **家具与产品设计师**
- **展览与活动设计师**

行业合作伙伴包括：**Adidas、Herman Miller、Accor Hotel Group**。

毕业生就职于：**AECOM、Area、IKEA**。
