# BA (Hons) 传播设计

## 课程概览

我们的 BA (Hons) Communication Design 学位在**概念设计**、**应用设计**和**创业实践**三个核心领域提供专业训练，通过多种媒介传达信息。课程运用设计思维转化和解读数据，将信息发展为合乎伦理、有意义且包容的传播设计体验。

你将获得**动态影像与动画**、**摄影**、**平面设计**（含插画）和**交互设计**（UX/UI/体验设计）方面的全面实践经验——在各种创意方法中建立专业技能，使你在毕业时极具就业竞争力。

课程在**苏格兰边境校区**授课，通过 **Go Global** 项目可前往欧洲、印度和美国的合作院校交流学习，或转至迪拜校区。

## 为什么选择这个专业？

- 涵盖传播设计全领域：平面设计、动画、摄影和 UX/UI
- 强调**伦理、包容和社会责任**设计
- **Go Global** 国际交流项目——前往欧洲、印度、美国合作院校或迪拜校区
- 专用 Mac 实验室、摄影棚和专业设计工坊
- 大三可选**行业实习**和**真实客户项目**
- 毕业年 **Design Showcase** 面向行业展示

## 入学要求

- **学术要求（大一标准）：** 苏格兰高中 BBBB / A-Level BCC + 数字作品集 / IB 29分 / BTEC Extended Diploma DMM
- **最低要求（公平入学）：** 苏格兰高中 BBBC / A-Level BCC / IB 29分
- **大二直入：** Advanced Highers BB / A-Level BBC / IB 30分
- **大三直入：** HND BB级（相关学科如计算机艺术、视觉传达、平面设计/插画/用户体验）
- **英语要求：** 雅思总分 6.0，各单项不低于 5.5
- **作品集：** 所有申请者须提交数字作品集——PDF格式，10-20张图片，展示研究、设计发展、技术技能和视觉传达能力

## 课程结构

### 第一年 — 基础

时尚、传播、营销与零售方向学生共同的基础学年，培养设计、可视化、摄影、视觉发展和设计思维的基础能力。

| 学期 | 课程 |
|------|------|
| 9月 | 设计语境与过程 |
| 9月 | 设计传达 |
| 1月 | 设计原型制作 |
| 1月 | 设计挑战 |

### 第二年 — 专业方向

引入字体设计、摄像、广告、动态图形和创意写作等专业主题。

**必修：**

| 学期 | 课程 |
|------|------|
| 9月 | 设计思维 |
| 9月 | 创意广告 |
| 9月 | 动态图形与内容创作 |
| 1月 | 设计研究方法 |
| 1月 | 实体数字体验与环境 |

**选修（1月 — 任选一门）：**

- 产品创新
- 趋势预测与媒体内容制作
- 设计模拟与建模
- 艺术化智能设计
- 设计中的循环经济

### 第三年 — 专业实践

发展专业技能，提供设计管理、时尚传播、交互设计、图形与插画等方向。

**必修：**

| 学期 | 课程 |
|------|------|
| 9月 | 专业实践 |
| 9月 | 动态跨媒体叙事 |
| 9月 | 艺术指导 |
| 1月 | 设计研究规划 |

**选修（1月）：**

- 真实项目
- 行业实习

### 第四年 — 毕业年

在导师指导下，通过两个学期的毕业项目整合所学技能。

| 时段 | 课程 |
|------|------|
| 9月 | 专业创意未来 |
| 1月 | 设计展示 |
| 全年 | 毕业设计项目 |
| 全年（选修） | 技术与创新 |
| 全年（选修） | 营销管理与战略 |

## 职业前景

毕业生在创意产业中从事多种角色：

- **平面设计师** — 品牌设计、编辑设计、包装、导视
- **UX/UI 设计师** — 数字产品与体验
- **艺术总监 / 创意总监** — 广告与媒体
- **动态图形设计师** — 动画与影视
- **品牌策略师** — 企业形象与传播
- **交互设计师** — 体验与沉浸式设计
- **视觉陈列师** — 零售与空间传播

*毕业生平均薪资：£24,000（毕业后15个月）。*
