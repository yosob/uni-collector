# BA (Hons) 时尚品牌与推广

## 课程概述

我们的 BA (Hons) Fashion Branding and Promotion 课程提供**时尚传播**、**营销**、**品牌建设**和**管理**的综合学习，专门针对全球时尚行业的需求而设计。

你将获得实践和创意设计技能的独特组合，同时全面了解时尚行业的商业层面。课程提供专业模块，涵盖传播设计、摄影、动态影像、视觉传达、时尚营销、品牌建设和消费者行为。

课程重点聚焦**可持续和包容性商业实践**。你将培养现场品牌活动管理和推广技能，运用分析和解读能力评估当前市场趋势并预测未来趋势。

课程在**苏格兰边境校区**授课，并有机会前往迪拜校区或欧洲、印度和美国的国际合作院校交换学习。

## 为什么选择这个课程？

- 时尚传播、营销、品牌建设和管理的独特组合
- 配备 Mac 和 PC 的**CAD 实验室**及**Adobe Creative Cloud**
- 设备齐全的**摄影工作室**用于时尚、人像和产品摄影
- **Go Global** 国际交流项目——在迪拜、欧洲、印度或美国学习
- 与**Louis Vuitton、W.L Gore**等品牌的行业实战项目
- 第三年可选行业实习

## 入学要求

- **学术要求（大一标准）：** 苏格兰高中 BBBB / A-Level BCC / IB 29分 / BTEC Extended Diploma DMM
- **最低要求（公平入学）：** 苏格兰高中 BBBC / A-Level BCC / IB 29分
- **大二直入：** Advanced Highers BB / A-Level BBC / IB 30分
- **大三直入：** HND BB级相关科目（零售管理、时尚设计与生产零售等）
- **英语要求：** 雅思总分 6.0，各单项不低于 5.5

## 课程结构

### 第一学年——基础（公共课程）

所有时尚、传播、营销与零售方向学生的公共基础学年。

| 学期 | 课程 |
|------|------|
| 9月 | 设计语境与过程 |
| 9月 | 设计传达 |
| 1月 | 设计原型制作 |
| 1月 | 设计挑战 |

### 第二学年——专业化方向

引入消费者行为、广告、数字媒体和排版等专业知识。

**必修课程：**

| 学期 | 课程 |
|------|------|
| 9月 | 设计思维 |
| 9月 | 创意广告 |
| 9月 | 动态图形与内容创作 |
| 1月 | 设计研究方法 |
| 1月 | 融合体验与环境 |

**选修课程（1月——选择一门）：**

- 艺术化智能设计
- 趋势预测与媒体内容制作
- 设计中的循环经济
- 产品创新
- 设计模拟与建模

### 第三学年——专业实践

通过时尚造型、创新创业、活动管理和视觉陈列培养批判性反思和分析技能。

**必修课程：**

| 学期 | 课程 |
|------|------|
| 9月 | 专业实践 |
| 9月 | 传承与当代奢侈品牌建设 |
| 9月 | 时尚创意指导 |
| 1月 | 设计研究规划 |

**选修课程（1月）：**

- 行业实习
- 实战项目

### 第四学年——荣誉学年

完成自主选择的荣誉项目，通常与行业合作伙伴协作。

| 时间段 | 课程 |
|--------|------|
| 9月 | 专业创意未来 |
| 1月 | 设计展示 |
| 全学年 | 荣誉项目 |
| 全学年（选修） | 营销管理与战略 |
| 全学年（选修） | 技术与创新 |

## 就业前景

毕业生在时尚行业从事多元化职业：

- **时尚品牌经理 / 营销经理**
- **创意总监 / 视觉陈列师**
- **时尚传播专家**
- **活动经理 / 品牌策略师**
- **时尚摄影师 / 内容创作者**

行业合作伙伴包括：**Crisis、Louis Vuitton、Malton Young and Borland、W.L Gore**。

*毕业生平均薪资：£24,000（毕业后15个月）。*
