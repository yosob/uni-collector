# BA (Hons) 时尚设计

## 课程概述

我们的 BA (Hons) Fashion 学位专注于培养你的创意和智识能力，为你在时尚设计领域——从奢侈时装品牌、高街品牌到独立设计师——的职业生涯奠定基础。你将创作从大众成衣到高级定制等各类服装。

你将系统学习**打版**和**服装制作**的理论与实践，将设计理念从图纸转化为成品服装。课程涵盖从高街时尚到成衣定制的多个细分市场，面向英国和全球市场，同时高度重视时尚行业的**可持续发展**、道德规范和多元化。

课程在**苏格兰边境校区**授课，地处苏格兰奢侈品纺织产业的核心地带，将传统工艺与当代时尚创新相结合。

## 为什么选择这个课程？

- **苏格兰第3名**——时尚与纺织专业（卫报大学指南 2026）
- 专用工作坊配备工业标准生产设备
- **Go Global** 国际交流项目——在全球合作大学学习
- 与奢侈品纺织制造商的紧密行业联系
- 最后一年**毕业展**直接面向业界展示
- 第三年可选行业实习和实战项目

## 入学要求

- **学术要求（大一标准）：** 苏格兰高中 BBBB / A-Level BCC + 数字作品集 / IB 29分 / BTEC Extended Diploma DMM
- **最低要求（公平入学）：** 苏格兰高中 BBBC / A-Level BCC / IB 29分
- **大二直入：** Advanced Highers BB / A-Level BBC / IB 30分
- **英语要求：** 雅思总分 6.0，各单项不低于 5.5
- **作品集：** 所有申请者必须提交数字作品集

## 课程结构

### 第一学年——基础（公共课程）

所有时尚、传播、营销与零售方向学生的公共基础学年，培养在当代时尚市场中的分析、评估和语境化能力。

| 学期 | 课程 |
|------|------|
| 9月 | 设计语境与过程 |
| 9月 | 设计传达 |
| 1月 | 设计原型制作 |
| 1月 | 设计挑战 |

### 第二学年——专业化方向

深入学习时尚方向，通过传统和非传统方法论探索打版、服装设计与制作。

**必修课程：**

| 学期 | 课程 |
|------|------|
| 9月 | 设计思维 |
| 9月 | 循环设计——可持续纺织与时尚概念 |
| 9月 | 时尚设计入门实践 |
| 1月 | 设计研究方法 |
| 1月 | 中级时尚设计实践 |

**选修课程（1月——选择一门）：**

- 产品创新
- 趋势预测与媒体内容制作
- 设计模拟与建模
- 艺术化智能设计
- 设计中的循环经济

### 第三学年——专业实践

通过独立学习、团队合作和外部项目进行跨学科项目和时尚行业合作。

**必修课程：**

| 学期 | 课程 |
|------|------|
| 9月 | 专业实践 |
| 9月 | 3D 数字时尚设计 |
| 9月 | 高级时尚设计实践 |
| 1月 | 设计研究规划 |

**选修课程（1月）：**

- 行业实习
- 实战项目

### 第四学年——荣誉学年

通过创作当代时装系列和完善作品集来巩固你的技能。

| 时间段 | 课程 |
|--------|------|
| 9月 | 专业创意未来 |
| 1月 | 设计展示 |
| 全学年 | 荣誉项目 |
| 全学年（选修） | 营销管理与战略 |
| 全学年（选修） | 技术与创新 |

## 就业前景

毕业生在时尚行业从事多元化职业：

- **时装设计师**——奢侈品牌、高街品牌或独立品牌
- **打版师 / 服装工艺师**
- **创意总监 / 时装插画师**
- **纺织设计师 / 买手**
- **时尚品牌顾问**

行业合作伙伴包括：**Johnsons of Elgin、W L Gore、Morton Young and Boreland、Halley Stevenson's**。

*毕业生平均薪资：£24,000（毕业后15个月）。*
