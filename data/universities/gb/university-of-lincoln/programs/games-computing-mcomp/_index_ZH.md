# 游戏计算综合硕士 — 林肯大学

## 课程概述

**游戏计算综合硕士（MComp Games Computing）**是一个四年制综合硕士课程，培养游戏设计、编程和开发方面的高级技能。第一至第三年遵循BSc课程，第四年提供专业主题的高级学习。

## 基本信息

| 项目 | 详情 |
|------|------|
| **校区** | Brayford Pool，林肯 |
| **学制** | 4年（全日制） |
| **学院** | 林肯计算机科学学院 |

## 入学要求

| 要求 | 详情 |
|------|------|
| **A Level** | BCC–BBC（104–112 UCAS积分） |
| **雅思** | 总分6.0（各单项不低于5.5） |

## 就业前景

高级游戏程序员、首席游戏设计师、软件架构师、AI专家、技术总监和研发工程师。
