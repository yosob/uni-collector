# 时尚设计文学学士（荣誉）— 林肯大学

> ⚠️ **该专业已停招。** 原官方课程页面已重定向至通用"Study with Us"页面。以下数据通过归档页面和第三方来源重建。

## 课程概述

林肯大学的**时尚设计文学学士（荣誉）（BA (Hons) Fashion）**是一门以创意与实践为导向的学位课程，鼓励学生通过时尚设计、服装结构、插画和情境研究等方面的实验来发展个人设计风格。该课程隶属于**林肯艺术与表演学院（Lincoln School of Fine & Performing Arts）**，在2015年英国全国学生调查（NSS）中排名**全英第3**。

课程将传统工艺技能与当代设计思维相结合，提供配备工业缝纫机、女帽设备、数码织物印花机、激光切割和3D打印等专业工作室。

## 课程结构

> ⚠️ 模块列表基于归档和第三方来源重建，无模块代码。

### 第一年（Level 4）— 基础建立

| 模块 | 描述 |
|------|------|
| **时尚设计基础** | 时尚设计基本原理、概念思维与创意过程 |
| **服装结构与裁剪** | 服装结构技术、打版与基础技术技能 |
| **时尚插画** | 时尚插画技巧与视觉传达 |
| **情境研究** | 时尚理论、历史与文化语境 |

### 第二年（Level 5）— 专业深化

| 模块 | 描述 |
|------|------|
| **时尚建筑** | 结构化与建筑化时尚设计方法、实验性构造 |
| **时尚解构** | 解构与重构技术、创新服装设计 |
| **表演服装设计** | 协作模块——参与制作团队，发展可迁移的协作技能 |
| **情境研究2** | 当代文化与产业中的时尚批判性研究 |

### 第三年（Level 6）— 专业转型

| 模块 | 描述 |
|------|------|
| **高级女帽制作** | 高级女帽与帽子制作技术 |
| **专业形象构建** | 专业身份构建、作品集展示与职业准备 |
| **毕业系列/毕业大秀** | 自主毕设项目，毕业大秀展示 |
| **情境研究3/毕业论文** | 独立研究论文 |
